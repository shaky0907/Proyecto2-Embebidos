list-device-regs

Synopsis

list-device-regs [root_of_search] "pattern"

Description

List all registers whose name matches pattern. If the optional argument root_of_search is set, the search is limited to this object and all its child objects. When not set, all objects in the simulation will be searched.

Simics Core
